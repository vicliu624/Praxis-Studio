# Praxis Studio Logo Candidate V4

Status: Candidate, pending user confirmation.

This candidate responds to feedback that V3 gained order but became too complex. V4 keeps the ordered governance semantics and removes almost everything else.

## Concept

The mark is a minimal Memory Cell:

- The amber open cell represents Praxis' controlled development boundary and studio craft.
- The three cyan nodes represent FACT, CANDIDATE and CONFIRMED memory.
- The open left side suggests intake and review before memory enters the governed system.
- The absence of document lines, inner rails and output nodes keeps the mark simple enough for navigation and app icon use.

## Design Rules

- One main amber shape.
- Three cyan nodes.
- No monogram target.
- No document icon.
- No graph mesh.
- No internal text lines.
