# Praxis Studio Logo Candidate

Status: Superseded candidate.

Superseded by: `docs/brand/praxis-logo-candidate-v2.md`

Reason: user feedback found the first mark too constrained by the letter `P` and visually closer to a `D`. The stronger direction is to keep amber governed flow and cyan memory nodes while moving away from monogram logic.

This candidate logo direction is a durable design record for the current Praxis Studio brand exploration. It is not confirmed project memory until accepted by the project owner.

## Concept

The mark is a disciplined graph-shaped `P`:

- The amber path forms the Praxis initial and reads as a controlled development flow.
- The cyan nodes represent local facts, candidate inference, and confirmed memory moving through governance.
- The small inner trace suggests Project Memory and Development Graph projection without turning the mark into a generic network diagram.
- The dark outline keeps the mark legible inside the existing dark desktop shell and at app-icon sizes.

## Usage Notes

- Use the mark alone in compact navigation, app icon contexts, and favicons.
- Use the compact navigation mark when the rendered size is around 24 to 32 pixels.
- Use the lockup where the project name must be visible.
- Keep the mark on dark or neutral backgrounds for now. A light-background variant should be created before use in documents or marketing surfaces.
- Do not treat this candidate as an accepted brand decision until the project owner confirms it.
