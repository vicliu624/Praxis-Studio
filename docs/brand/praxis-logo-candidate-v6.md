# Praxis Studio Logo Candidate V6

Status: Confirmed.

Confirmed by: project owner.

Decision date: 2026-06-22.

Authority document: `docs/brand/praxis-logo.md`

This candidate responds to feedback that V5 became too abstract. V6 restores a readable project-memory object while keeping the mark disciplined and simple.

## Concept

The mark is a Memory Document Graph:

- The amber document represents durable Project Memory: normalized docs plus Git history.
- The three cyan nodes represent FACT, CANDIDATE and CONFIRMED memory.
- The amber rails show governed intake into memory, rather than freeform graph drawing.
- The folded corner makes the memory/document meaning visible without a legend.

## Design Rules

- One readable document shape.
- Three governed memory nodes.
- Short controlled rails only.
- No monogram target.
- No organic orbit.
- No general-purpose mesh graph.
