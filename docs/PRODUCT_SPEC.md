# Praxis Studio Product Spec

Praxis Studio 是一个以开发图谱和项目记忆为中心的 AI-native Product Development IDE。

核心循环：

```text
产品构想 → 需求拆解 → 架构映射 → 开发图谱 → 模块进度 → 胶水进度 → Agent 施工 → 验证反馈 → 项目记忆
```

产品原则：

1. 树表达归属。
2. 图表达关系。
3. 节点和边都是可选择对象。
4. Chat 绑定当前节点、边或子图。
5. 节点有模块进度，边有胶水进度。
6. Agent 先解释，再计划，再执行。
7. 每次变更都形成项目记忆。
